# template snakemake-with-R
 Snakemake template for running R scripts

Use this template as follows: 

Step 1: Activate conda and snakemake

In terminal, run

```{r eval=FALSE}
$ conda activate base
$ conda activate snakemake
```

Step 2: Go to the 

```{r eval=FALSE}
$ cd /path/to/snakemake-with-R
```
